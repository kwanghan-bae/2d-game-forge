inflation-rpg 개인정보처리방침

(draft v0.1 — 출시 전 법무 검토 필요)

1. 수집하는 정보

본 게임은 사용자의 개인정보를 직접 수집하지 않는다. 다음의 제3자 SDK가 광고 송출 및 결제 처리를 위해 일부 정보를 수집한다:

  • Google AdMob — 광고 송출. 광고 ID, 기기 정보, IP 주소 등을 Google 의 정책에 따라 수집한다. 자세한 내용: https://policies.google.com/technologies/ads
  • ONE store IAP SDK — 인앱결제 처리. 결제 토큰, 상품 ID 등을 ONE store 의 정책에 따라 수집한다. 자세한 내용: https://www.onestore.co.kr/userpolicy/privacy

2. 게임 진행 데이터

게임 진행 데이터 (캐릭터 레벨, 균열석, 던전 진척 등) 는 사용자 기기 내부에만 저장되며 외부로 전송되지 않는다.

3. 광고 제거 (인앱결제)

"광고 제거" 인앱결제 구매 시, ONE store 결제 토큰만 기기 내부에 저장된다. 결제 영수증 검증은 ONE store SDK 의 client-side API 만 사용한다.

4. 데이터 보관 기간

모든 게임 데이터는 사용자가 앱을 삭제할 때까지 기기에 보관된다. 별도 서버 보관은 없다.
